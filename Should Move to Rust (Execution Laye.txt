  3. Delta Clamping (Safety) 🚨 Critical

  Current Location: test_interpolation_extrapolation.py (Python)

  Why Rust:
  - Safety-critical - prevents over-warping artifacts
  - Must be deterministic (Python could have GC pauses)
  - Called before every granular synthesis operation
  - Prevents "Uncanny Valley" artifacts

  Rust Implementation:
  pub struct SafetyClamp {
      max_safe_warp: f32,
  }

  impl SafetyClamp {
      pub fn clamp_target(&self, target: &Vector17D, anchor: &Vector17D) -> Vector17D {
          let distance = target.distance_to(anchor);
          if distance > self.max_safe_warp {
              // Apply clamp (deterministic)
              let direction = target - anchor;
              anchor + (direction.normalized() * self.max_safe_warp)
          } else {
              *target
          }
      }
  }

  ---
  Should Stay in Python (Logic Layer)

  1. Intent Decomposition ✅ Keep in Python

  Why:
  - Cognitive Logic: Which modalities for which intents?
  - Semantic Reasoning: Alarm = Whistle + Rattle + Whistle
  - Can be slow: Not in real-time loop
  - Needs flexibility: Easy to adjust strategies

  2. Acoustic Algebra (Virtual Target Calculation) ✅ Keep in Python

  Why:
  - Semantic Strategy: What does "50% aggression" mean acoustically?
  - Configuration: Easy to tweak parameters
  - Called infrequently: Once per intent, not per sample
  - Rust should handle the numerical computation, Python defines the strategy

  3. Navigation Strategy Planning ✅ Keep in Python

  Why:
  - High-level planning: Which route to take?
  - Context-aware: Depends on environmental state
  - Not time-critical: Planned once, then executed
